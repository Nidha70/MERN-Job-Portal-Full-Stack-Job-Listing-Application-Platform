# MERN-Job-Portal-Full-Stack-Job-Listing-Application-Platform
Full-stack Job Portal built with MERN stack allowing recruiters to post jobs and users to search and apply with a modern UI built using Tailwind CSS and shadcn/ui.
## Job Portal Web Application

This project is a full-stack Job Portal built using the MERN stack. 
The platform allows recruiters to post job listings and job seekers to browse and apply for jobs through a modern and responsive interface.

### Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- shadcn/ui
- React Hooks
- REST APIs

### Features
- User authentication
- Job posting and job listing
- Job search functionality
- Job application management
- Responsive UI design
